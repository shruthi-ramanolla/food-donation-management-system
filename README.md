# 🍲 Food Donation Management System

## 📌 Project Overview
The Food Donation Management System is a web-based platform developed to reduce food wastage by connecting food donors with NGOs and needy people.  
This system helps manage food donations efficiently and ensures safe and timely distribution.

---

## 🎯 Objectives
- Reduce food wastage.
- Provide an easy-to-use platform for donors and NGOs.
- Track donation status.
- Ensure proper food distribution.

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## 👥 Modules

### 1️⃣ Donor Module
- Register/Login
- Add food donation details
- View donation status

### 2️⃣ Admin Module
- Verify donation requests
- Assign NGOs
- Manage system data

### 3️⃣ NGO Module
- Accept donation request
- Update pickup and delivery status

### 4️⃣ Beneficiary Module
- Receive food from NGO

---

## 🔄 System Workflow

Donor → Website → Admin → NGO → Beneficiary

---

## ✨ Features
- Simple and user-friendly interface
- Real-time status updates
- Organized donation tracking
- Helps reduce food wastage

---

## 🚀 How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Or run using Live Server in VS Code.
